# API 서버 Result 값 정의

|Code|Description|
|----|-----------|
|0|Transaction succeed|
|-1|No record|
|-2|Invalid parameter|
|-3|Internal server error|
|-4|Internal client error|

|MsgCode(Upper) 2byte|MsgCode(Lower) 2byte|
|--------------------|--------------------|
|2 Byte|2 Byte|
|ex) 0x00010002|

|MsgCode|Description|
|-------|-----------|
|0x0001|Reserved|
